# Shahram Shafiq | Personal Portfolio

A personal portfolio website built as a single-file static site (pure HTML, CSS, and JavaScript). No frameworks, no build step.

Live at: [shahramshafiq-portfolio.vercel.app](https://shahramshafiq-portfolio.vercel.app)

## Features

- Animated 4-pointed sparkle particle background on the hero section
- Scroll-reveal animations on all sections
- Hover effects on skill pills, project cards, and navigation links
- Working contact form via Web3Forms (no backend needed)
- Mobile hamburger navigation menu
- Fully responsive across all screen sizes
- Deployed on Vercel
